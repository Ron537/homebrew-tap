# Ron537's Homebrew Tap

Homebrew casks for [Ron537](https://github.com/Ron537)'s macOS apps.

```bash
brew tap ron537/tap
```

## Apps

| App | Install | Repo |
|-----|---------|------|
| **Cliplex** — fast, private clipboard manager (history, snippets, actions, instant search) | `brew install --cask ron537/tap/cliplex` | [Ron537/Cliplex](https://github.com/Ron537/Cliplex) |

More apps coming.

## Notes

Apps are ad-hoc signed (free, not notarized), so on first launch macOS may say
it "could not verify" the app — approve it once via **System Settings → Privacy
& Security → Open Anyway** (each cask's `caveats` explains this).
